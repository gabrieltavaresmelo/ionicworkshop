### COMANDOS:
```bash
$ sudo npm install -g ionic cordova
$ ionic start mySideMenu sidemenu --v2
$ ionic g page cursodetails
$ ionic g provider CursoService
$ ionic serve --lab
```

### EXECUCAO:
```bash
$ ionic platform add android
$ ionic run android
```

### RESOURCES:
```bash
$ ionic resources
$ ionic resources --icon
$ ionic resources --splash
```
